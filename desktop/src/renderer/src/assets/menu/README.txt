TAOM RASMLARI
=============

Bu papkaga taom rasmlarini joylashtiring. Fayl nomi quyidagicha bo'lishi kerak
(menyu elementiga shu nom bo'yicha bog'lanadi):

  osh.jpg              -> Osh
  lagmon.jpg           -> Lag'mon
  shashlik.jpg         -> Shashlik
  achchiq-chuchuk.jpg  -> Achchiq-chuchuk
  sezar.jpg            -> Sezar
  choy.jpg             -> Choy
  cola.jpg             -> Coca-Cola
  pivo.jpg             -> Pivo (0.5)

Format: jpg / jpeg / png / webp. Tavsiya: kvadratga yaqin, ~600x600 px.

Rasmni almashtirgandan keyin ilovani qayta yig'ing:
  npm run build:mock --workspace=@hardweb-pos/desktop
  (keyin installer uchun) npx electron-builder --win   (desktop papkasida)
